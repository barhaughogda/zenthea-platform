## Explicit Prohibitions
The following activities remain explicitly prohibited:
- **Ambient listening**: No continuous audio capture or processing.
- **Covert capture**: All data capture must be visible and explicitly authorized.
- **Retroactive consent**: Consent must be obtained prior to data capture.
- **Voice-triggered execution**: Voice commands shall not trigger system-state changes without secondary confirmation.
